# 🎲 Two Player Dice Rolling Game (Python)

This is a simple command-line based dice game built using Python for two players.

## 🚀 Game Description
- The game is played between two players.
- Each player rolls a dice in every round.
- The dice generates a random number between 1 and 6.
- The scores are added after each round.
- At the end of all rounds, the player with the highest score wins.

## 🎯 Features
- Two-player gameplay
- Random dice rolls
- Multiple rounds support
- Score tracking system
- Winner declaration at the end

## 🛠️ Technologies Used
- Python
- Built-in module: random

## ▶️ How to Run

1. Make sure Python is installed
2. Run the following command:

```bash
python dice_game.py
